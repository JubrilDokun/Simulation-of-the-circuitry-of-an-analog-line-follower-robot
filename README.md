# Simulation-of-the-circuitry-of-an-analog-line-follower-robot
The circuit simulated controls the left and right motors of a line-follower robot using the outputs from op-amps which compare voltages from photodiodes and prefixed potentiometers.  
This circuit simulation was done on Autodesk Tinkercad (link provided below).
